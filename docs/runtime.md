# Runtime notes

- Sanitizers: default Debug build uses `-fsanitize=${SANITIZERS_LIST}`; adjust via `-DSANITIZERS_LIST="address;undefined;thread"` or disable with `-DENABLE_SANITIZERS=OFF`. Avoid mixing `thread` (TSan) with ASan in the same list.
- LeakSanitizer: in this sandbox LSan crashes under ptrace; use `ASAN_OPTIONS=detect_leaks=0` (the `dev.sh` helper sets this via `LEAKS` env var).
- Coverage: enable with `-DENABLE_COVERAGE=ON` (Clang) then run `cmake --build <dir> --target coverage` to produce `coverage/html/`.
- Presets: Conan preset `conan-debug` is generated by `conan install`; native builds default to `build` dir.
- Tests carry labels: `unit;sanitizers` to help future CTest/CI filtering.
